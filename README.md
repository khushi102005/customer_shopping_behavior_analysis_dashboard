# Customer Shopping Behavior Analysis Dashboard
📊 Customer Shopping Behavior Analysis

📌 Overview

This project analyzes customer shopping behavior using transactional data to generate actionable business insights. It covers the complete data analytics workflow — from data cleaning and exploration to SQL-based analysis and interactive dashboard creation.
The objective is to understand customer segmentation, spending patterns, subscription behavior, product performance, and revenue contribution to support data-driven decision-making.

📂 Dataset

Total Records: 3,900 transactions
Total Features: 18 columns
Key Attributes:
Customer Demographics (Age, Gender, Location, Subscription Status)
Purchase Details (Item, Category, Purchase Amount, Season, Size, Color)
Behavioral Data (Discount Applied, Promo Code Used, Previous Purchases, Review Rating, Shipping Type)

Data Cleaning:
Handled missing values in Review Rating
Standardized column names
Created new features such as age groups and purchase frequency

🛠️ Tools & Technologies

Python (Pandas, NumPy) – Data cleaning & EDA
SQL (PostgreSQL/MySQL/SQL Server) – Business queries & analysis
Power BI – Interactive dashboard creation
MS PowerPoint / Gamma – Final project presentation

2)Data Cleaning & Feature Engineering

3)SQL Analysis
  Revenue by gender

  High-spending discount users

4)Dashboard Development

5)Reporting & Presentation
